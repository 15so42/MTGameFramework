# 此仓库引用于UnityGameFramework的StarForce示例项目，在StarForce的基础上清除StarForce项目资源文件，保留常用工具类，实体类，并更改命名空间，使其能作为通用模板进行游戏开发。
项目中画布更改为1080p,使用通用场景加载流程。
其中有一个RagdollPlayer预制体作为玩家，在ProcedureMain中的SurvivalGame类中调用GameEntry.Entity.ShowEntity进行生成。
RagdollPlayer仅持有一个控制脚本，保留于项目中作为示例，使用时删除RagdollPlayer相关文件并替换为新的Player即可。
